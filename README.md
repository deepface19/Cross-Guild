## Installing and running
```
git clone https://github.com/deepface19/Cross-Guild.git
```
##### Setup Hardhat/Contract
```
cd packages/hardhat 
yarn install
yarn deploy:luksoL16 
```
##### Setup Frontend
```
cd packages/frontend
npm install 
npm start [localhost:3000]
```
